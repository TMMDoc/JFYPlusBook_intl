Output settings

Save layout NC and reports – Aktivieren Sie diese Option, damit Layout-NC und Berichte gespeichert werden.

Save layout file – Aktivieren Sie diese Option, damit die Layoutdatei gespeichert wird.

Output simulation preview in reports – Aktivieren Sie diese Option, um die Werkzeugsimulationsvorschau zusammen mit den anderen Ausgaben zu generieren. Das ist eine kurze GIF-Animation für das Laser-/Stanzrüsten.

Do not delete program outputs on layout/job completion – Wenn Sie diese Option aktivieren, wird der Löschvorgang für die Programmausgaben übersprungen, wenn ein Job/Layout als abgeschlossen markiert wird.

Nest settings

Mix parts from multiple Jobs – Unterstützt das Schachteln im gemischten Modus. Teile aus mehreren Jobs sollten für eine bessere Packeffizienz miteinander geschachtelt werden.

Do not auto-release layouts – Dadurch wird verhindert, dass Layouts für die Produktion verfügbar sind, bevor alle Layouts in freigegebenem Zustand sind.

Re-nesting packing cut-off – Dieser Wert wird verwendet, um zu entscheiden, ob ein geschachteltes Layout in der Genehmigungswarteschlange gehalten und neu geschachtelt wird, wenn neue Teile geplant werden, oder ob es für die Produktion freigegeben wird und die zugehörigen Ausgaben in den Ausgabeordner exportiert werden.

Interactive sheet-name prefix – Dieser Abschnitt wird verwendet, um interaktives Schachteln zu unterstützen, bei dem der Benutzer mehr Kontrolle über die geschachtelten Teile und das Schachtelergebnis hat. Verwenden Sie für die interaktiv geschachtelten Layouts ein anderes Namensschema als für die automatisch geschachtelten.

Update sheet inventory on nest (un)release – Wenn diese Option aktiviert ist, wird der Tafelbestand aktualisiert, wenn ein Nest an eine Maschine gesendet wird. Außerdem kehrt die Tafel in den Bestand zurück, wenn das Nest von der Maschine zurückgerufen wird.

Automate remnant inventory management – Wenn dieser Schalter gewählt ist, automatisiert die Software die Restbestandsverwaltung. Wird er nicht gewählt, wird die Resteverwaltung interaktiv.

Auto-refresh Out-of-date layouts - Wenn diese Option aktiviert ist, aktualisiert die Software alle veralteten Layouts, die mit einem Teil verknüpft sind, nachdem die Schneidprogramme berechnet wurden.

Automate twin-line nesting - Aktivieren Sie diese Option, um die TwinLine-Automatisierung zu aktivieren.

Pull workflow settings

late:SH.UsePullWorkflow[Use 'Pull workflow'] – Diese Option ist hilfreich für Benutzer, die ein hochautonomes Planungs- und Schachtelsystem benötigen. Wenn diese Option aktiviert ist, plant Praxis das Schachteln bei Bedarf.

Nests per station – Mit diesem Wert wird festgelegt, wie viele Layouts pro Station erstellt werden sollen.

Maximum sheets per nest – Dieser Wert wird verwendet, um die maximale Tafelanzahl für das eindeutige Nest zu steuern. Wenn es zum Beispiel einen Auftrag mit einem großen Teil (das die gesamte Tafel füllt) gibt, das in großen Mengen benötigt wird (z. B. 500), erstellt Praxis im Pull-Modus nicht mehr ein einziges Nest mit 500 Kopien. Stattdessen werden 10 Kopien pro Nest erstellt, wobei 10 das Maximum ist.

Bend programs per station – Mit diesem Wert wird festgelegt, wie viele Biegeprogramme pro Station erstellt werden sollen.

Mark the task completed, when the program outputs are removed – Aktivieren Sie diese Option, damit ein Layout als erledigt markiert wird, wenn die Ausgaben vom Sollausgabezielort entfernt werden.

Spreadsheet import settings

CAD lookup directories – Praxis wird versuchen, den Teileeintrag, der in der importierten CSV-Datei erwähnt wird, aus der Teilebibliothek zu laden. Wenn das Teil jedoch nicht in der Teilbibliothek verfügbar ist, wird versucht, die Teile ausgehend von der in diesem Dialog festgelegten Verzeichnisreihenfolge aus Suchverzeichnissen abzurufen.

Use recursive CAD search – Aktivieren Sie diese Option, damit die Suche für aufeinanderfolgende Ergebnissen wiederholt wird.

Create a blank part for the missing CAD files – Wenn Teile in der Praxis-Bibliothek oder in den Nachschlageverzeichnissen nicht vorhanden sind, erstellt Praxis für diese nicht verfügbaren Teile ein leeres Teil.

Do not show the New Job dialog if the spreadsheet has no error – Wenn CSV-Einträge sauber sind, d. h. keine Nichtübereinstimmung mit Daten aus der DB vorliegt oder alle Felder korrekt aufgelistet sind, zeigt Praxis das Jobübersichtsfenster nicht an und verarbeitet den Job, sobald die CSV importiert oder abgelegt wird.

Prefer Raw-Material column over Material + Thickness – Unterstützte Tabellenblatttypen:

  1. Muster 1: Teilename, Menge, Fälligkeitsdatum, Priorität, Rohmaterial

  2. Muster 2: Teilename, Menge, Fälligkeitsdatum, Priorität, Material, Dicke

  3. Muster 3: Teilename, Menge, Fälligkeitsdatum, Priorität, Material, Dicke, Rohmaterial

Wenn diese Option nicht aktiviert ist, wird Muster 1 als Standard-Tabellenblatttyp betrachtet. Wir unterstützen auch Muster 3, falls das CSV-Format vom Typ Muster 3 ist.

Wenn diese Option aktiviert ist und CSV vom Typ Muster 2 ist, leitet Praxis das Rohmaterial aus den CSV-Feldern für Material und Dicke ab.

Spreadsheet output settings

Export Job/Layout completion report (.csv) – Aktivieren Sie diese Option, um den CSV-Export zu aktivieren, wenn ein Layout oder ein Job abgeschlossen ist. Die abgeschlossenen Berichte werden an dem unter Werk → Einstellungen → Bericht angegebenen Berichtszielort gespeichert.
