Import

Import

Output settings

Save flat pattern - Aktivieren Sie diese Option, damit das Flachmuster eines importierten Teils gespeichert wird.

Format for flat pattern – Siehe JFY Core-Einstellungen. Flat pattern destination - Das ist der Ort, an dem das Flachmuster gespeichert wird.

Import settings

Units for DXF files – Siehe JFY Core-Einstellungen. Material lookup keys - Sie können mehrere Sucheingaben zuweisen, indem Sie ein Komma als Trennzeichen verwenden. Wenn ein Teil importiert wird, verwendet Praxis diese Schlüsselwörter zur Suche nach Materialwerten. Die gesuchten Werte werden dann zum Lösen des Rohmaterials verwendet. Die Originalwerte werden in den sonstigen Eigenschaften des Teils gespeichert.

Revision lookup keys - Beim Importieren eines Teils wird der Überarbeitungswert aus der Rohteildatei (der zu importierenden Teiledatei) mit dem im Repository vorhandenen Überarbeitungswert verglichen. Wenn die Überarbeitung identisch ist, wird der Import abgebrochen. Die benutzerdefinierte Feldzuordnung wird nun auch für die DXF-Dateien unterstützt. Für die DXF-Dateien werden die benutzerdefinierten Felddaten aus der Textebene in der Zeichnung oder einer PPI-Anlage extrahiert.

Part name keys - Diese Option funktioniert ähnlich wie die Material- und Überarbeitungsschlüsselwörter. Legen Sie ein oder mehrere Schlüsselwörter für Teilenamen fest, um den anstelle des Dateinamens PPI-Teilenamen als Teilenamen zu verwenden.

Thickness lookup keys - Sie können mehrere Sucheingaben zuweisen, indem Sie ein Komma als Trennzeichen verwenden. Wenn ein Teil importiert wird, verwendet Praxis diese Schlüsselwörter zur Suche nach Werten für die Dicke. Die gesuchten Werte werden dann zum Lösen der Dicke verwendet. Die Originalwerte werden in den sonstigen Eigenschaften des Teils gespeichert.

Thickness comparison fudge - Dies wird verwendet, um das Rohmaterial nachzuschlagen, wenn ein Teil importiert wird. Praxis versucht, das Rohmaterial zu lösen, nachdem der Materialname und die Dicke analysiert wurden. Weitere Informationen finden Sie im Thema [Thickness fudge] unten.

Identify shapes after a flat is imported - Wenn ein Teil importiert wird, können Innenformen innerhalb des Teils automatisch erkannt und identifiziert werden.

Shape identification fudge - Der Fudge-Wert für die Formenidentifizierung kann auf folgenden Ebenen eingestellt werden:

  1. Globale Ebene - Dieser Wert wird als Standard für alle Formen verwendet.

  2. Ebene einzelner Formen - Der Standardwert kann überschrieben werden, indem der Fudge-Wert auf der Ebene einzelner Formen festgelegt wird. Wenn Sie die Form im Dialog Formbibliothek auswählen, wird der Fudge-Wert angezeigt. Standardmäßig ist er auf 0 festgelegt, und für solche Formen wird der globale Standardwert verwendet.

Rotate flat horizontally - Im Allgemeinen sind Schneidmaschinentische rechteckig und breiter. Um die Machbarkeit des Rüstens zu verbessern, drehen Sie das Teil nach dem Importieren entlang der Teilbreite.

Part rotation threshold - Legen Sie dies als Mindestwert fest, ab dem die Drehung zulässig ist. Jedes Teil, das kleiner als der eingestellte Wert ist, wird nicht gedreht.

Misc. settings

Enable pre-import external tools - Aktivieren Sie die Option für den Vorabimport von Fremdwerkzeugen, um Creo Reader zu aktivieren. Praxis lädt die vorab extrahierten Familieninstanzen des abgelegten Teils hoch. Bitte beachten Sie, dass die Instanz-Extraktion während des Teile-Uploads durchgeführt wird (als dem Import vorgeschalteter Vorgang), weshalb die Creo-Software auf dem Computer installiert werden muss, von dem das Teil hochgeladen wird.

Ignore drawing colors in Flat thumbnails for clarity - Aktivieren Sie diese Option, um graustufenschärfere Miniaturansichten zu generieren. Die Schichtfarben und Texteinheiten werden aus Gründen der Bildklarheit ignoriert.

Use only one Engine for all import tasks - Wenn aktiviert, wird nur ein Engine-Kern für wichtige Aufgaben verwendet.

Fudge-Wert für die Dicke

Wenn eine bestimmte Blechdicke nicht im Bestand verfügbar ist, wählt Thickness comparison fudge den nächstgelegenen Blechdickenwert aus, indem der Wert für die Dicke der Bleche im Bestand vergleicht.

Nehmen wir zum Beispiel Folgendes an:

  1. Wir haben ein Teil mit einer Blechdicke von 1,6 mm und dem Material Stahl.

  2. Der Thickness comparison fudge-Wert beträgt 0,1 mm.

  3. Im Bestand gibt es Stahlbleche mit einer Dicke von 1,5 mm, 1,55 mm und 1,7 mm.

    JFY Plus wählt ein Blech mit dem nächstgelegenen Dickenwert aus, nachdem der Thickness comparison fudge-Wert (0,1 mm) mit dem importierten Artikel (von 1,6 mm) verglichen wurde. In diesem Fall werden 1,55 mm ausgewählt.
