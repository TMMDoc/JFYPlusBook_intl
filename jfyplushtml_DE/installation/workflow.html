Workflow

Workflow

Dieses Kapitel gibt Ihnen einen grundlegenden Überblick über die Arbeitsschritte:

1. System einrichten

Bevor Sie beginnen, müssen Sie JFY Plus so konfigurieren, dass es Ihr Werk digital abbildet. Sie müssen Maschinen einrichten, Material hinzufügen, Werkzeuge einrichten und Benutzer konfigurieren.

2. Teile importieren und bearbeiten

Sie importieren Ihre Teile in die Teilebibliothek. Die Technologie wird Ihren Teilen, die Sie mit den unten gezeigten Icons überprüfen können, sofort hinzugefügt.

== 3. Auf Produktionsprobleme prüfen

Mit einem Rechtsklick auf das Teil können Sie alle Werkzeuge bearbeiten.

== 4. Jobs definieren

Um Teile zu Fertigungsaufträgen zu kombinieren, wählen Sie sie in der Teilebibliothek und erstellen Sie einen Job. Erstellen Sie den Job mit einem Rechtsklick.

== 5. Jobs Maschinen zuweisen

Wenn Sie mehr als eine Maschine haben, können Sie die Maschine wählen, die den Job übernehmen soll.

6. Filter verwenden, um Teile zu finden

Es gibt leistungsstarke Filteralgorithmen,, mit denen Sie jedes Teil Ihrer Kunden finden können.

7. Teil an Maschinen freigeben

Sie können sämtliche Jobs jederzeit ändern, bevor Sie sie an die Maschine freigeben. Um Ihre Teile zu produzieren, müssen Sie die Teile Maschinen zuweisen, dann wird der NC-Code an die Maschine übertragen.

== 8. Teile produzieren

Nach Freigabe des Speicherorts des Standardordners ODER Freigabe des Standorts des Code-Absenders senden Sie sie an Ihre Maschine, produzieren Sie sie und markieren Sie sie als abgeschlossen.
