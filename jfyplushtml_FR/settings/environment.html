Environnement

Environment settings

Program default units - Voir les réglages JFY Core. Decimal places to display - Voir les réglages JFY Core. Decimal accuracy in export/import - Cette valeur permet de définir le nombre exact de décimales lors de l’importation ou de l’exportation.

Display part toolings in groups - Utilisez cette option pour sélectionner le groupe de montage permettant d’afficher l’état de l’équipement dans la bibliothèque de pièces.

Captions for command-bar icons - Voir les réglages JFY Core. Display Job status labels - L’activation de cette option affiche le nombre de pièces qui se trouvent dans une tâche à la fin de la barre d’état de la tâche. Lorsque cette option est désactivée, le nombre de pièces est supprimé et la couleur d’affichage du processus est la même.

Display nested parts by job-order color - Cette option affiche chaque pièce d’une tâche d’une couleur différente. La désactivation de cette option affiche toutes les pièces dans la même couleur grise.

Display the duration in time-span format (HH:MM:SS) - Cochez cette option pour afficher la durée au format HH:MM:SS. Lorsque cette option est activée, la durée est affichée dans le format sélectionné dans les détails du layout et les rapports de tâche/layout. En raison des contraintes d’espace, la durée est affichée dans un format de secondes plus compact, sur les cases de layout.
