Filtre de bibliothèque de pièces

Filtre de bibliothèque de pièces

Filtrer les pièces par état de l’équipement

Sélectionnez le champ de recherche Etat de l’équipement pour filtrer les pièces en fonction de l’état de l’équipement. La liste déroulante affiche les états sous forme de cases à cocher colorées et de machines : blanc signifie

OK, Orange signifie Erreur et Jaune est un état d'avertissement. Le bouton Tout appliquer (&) est activé dans ce cas pour appliquer la condition de filtre à toutes les machines sélectionnées.

Filtre Description

Affiche les pièces qui présentent au moins une erreur d’outillage.

Affiche les pièces qui ne peuvent pas être coupées à l’aide de la machine TruLaser 3030.

Coupe impossible : Ces pièces ne peuvent être coupées sur aucune machine de coupe (noter la sélection de « & » ici).

Erreurs de pliage : Les pièces ne peuvent pas être produites car il n’y a pas d’outillage de pliage valide.

Pliage à plat UNIQUEMENT : Ces pièces ne peuvent être traitées que parce qu’il existe un outillage de pliage à plat. Celles-ci ne peuvent être traitées sur aucune des presses plieuses.

Filtrage des pièces par support de pliage

Sélectionnez le champ Supports de pliage pour rechercher des pièces par support de pliage.

  • La liste déroulante, pour ce champ, affiche la liste des outils de pliage. Sélectionnez un outil pour trouver la liste des pièces à l’aide de cet outil.

  • La liste d’outils dans la liste déroulante peut être réduite en saisissant le nom d’outil, le type d’outil (poinçon, col de cygne, rayon, etc.) et la description de l’outil.

  • Sélectionnez plusieurs outils et appliquez le modificateur TOUS (&) pour répertorier toutes les pièces à l’aide des deux outils.

  • Sélectionnez une pièce et double-cliquez pour ouvrir le détail de la pièce afin de mettre en évidence d’autres outils utilisés dans la pièce pour TOUTES LES presses plieuses. La description de l’outil est mise à jour avec les détails du support.

  • Double-cliquez sur la machine pour mettre en évidence les outils de pliage utilisés dans la pièce sélectionnée pour la machine sélectionnée.

En plaçant la souris sur le bouton du filtre de support de pliage, vous affichez l’info-bulle avec les icônes d’outils sélectionnés. Un crochet autour du nombre d’outils indique la condition de correspondance « TOUS » tandis qu’une parenthèse indique N’IMPORTE QUELLE correspondance.

Filtrer les pièces par outil utilisé

Ceci est identique au filtrage basé sur les supports utilisés, sauf :

  • Les icônes d’outil sont dessinées dans une taille proportionnelle (pas exactement aux proportions).

  • Lorsque l’aperçu de l’outillage est ouvert, déplacez la souris sur les outils en surbrillance pour mettre en évidence l’outillage correspondant dans le volet d’aperçu.

Filtrer les pièces par état

Les pièces de la bibliothèque peuvent désormais être filtrées par état. Les options affichent la liste de TOUS les états applicables, ainsi la liste change en fonction de la liste des pièces de la bibliothèque et de leurs états actuels. Choisissez Matière manquante pour lister les pièces avec un état de matière non résolu.

Filtrer les pièces par codes d’erreur

En combinaison avec l'état et les machines, des codes d’erreur de montage spécifiques peuvent désormais être utilisés dans la recherche par état de l’équipement.

L’image ci-dessous montre le scénario de recherche : Erreurs de montage Surcharge pour la machine 5170 6A.

La liste d’erreurs affiche les erreurs applicables pour la machine et la sélection de l’état. Elle s’actualise lors de modification de la machine et de la sélection d’état pour effectuer une recherche plus précise. Donc, si vous sélectionnez l’état d’avertissement pour une machine de poinçonnage, seuls les avertissements de poinçonnage disponibles sont affichés dans la liste des erreurs.

Filtre d’état masqué

Souvent, les montages de pièces sont masqués en raison d’une géométrie ou d’une épaisseur surdimensionnée. Le montage est également masqué s’il est spécifiquement refusé par une trajectoire CAO. JFY Plus affiche la raison du masquage dans l’info-bulle d’outillage. En outre, ces pièces peuvent être recherchées à l’aide du champ Etat masqué nouvellement ajouté dans le menu de recherche Etat de l’équipement.
