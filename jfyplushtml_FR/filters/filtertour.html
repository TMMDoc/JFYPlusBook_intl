Tour des filtres

Filtrer les options de page

  • Les filtres sont utilisés pour rechercher pièce, statut/état de montage, faisabilité de la production, outils de poinçonnage/pliage utilisés, statut de tâche, tâche par client, priorité de tâche, statut de layout, etc., en utilisant des éléments de recherche filtrant.

    • Filtre par défaut : Pièce / Tâche / Layout ont leur propre filtre par défaut. L’administrateur de site / les programmeurs peuvent y ajouter leurs propres listes de filtres requis.

    • Condition de filtre actuelle : En fonction de l’option de filtre sélectionnée ou de la condition ajoutée, la page s’affiche en fonction du filtre.

    • Condition de filtre : La condition de filtre varie en fonction du champ.

    • Eléments de filtre : Le champ élément de filtre change en fonction de la pièce, de la tâche ou du layout.

Filtrer les ordres

Ordre Raccourcis Description

Effacer la sélection +

Ctrl+X

Désélectionne tous les choix sélectionnés et actualise la vue des éléments de filtre.

Sélection en surbrillance

Ctrl+A

Sélectionne tous les choix en surbrillance. Cette commande reste désactivée si la mise en surbrillance d’élément n’est pas applicable pour le champ sélectionné.

Tout faire correspondre

&

Applique TOUS LES (vs UN DES) choix sélectionnés à la correspondance de filtre.

Epingler / Désépingler le menu

Ctrl+Down/Up

Lorsqu’il est épinglé, le menu déroulant reste ouvert même lorsque le focus actuel passe à une autre fenêtre.

Fermer le menu

Esc

Ferme le menu déroulant.

Tout faire correspondre

* L’ordre de filtre Match all est utilisé pour filtrer en fonction de tous les éléments sélectionnés. - Seules les pièces qui correspondent à tous les filtres sont affichées lorsque vous cliquez sur Match all.

Basculer l’opacité

  • L’ordre de filtre Toggle opacity est utilisé pour afficher les pièces derrière les éléments de filtre.

    • Cliquez sur Toggle opacity pour baisser l’opacité des éléments de filtre et afficher les pièces cachées derrière. Cliquez à nouveau sur ui:Str.ToggleOpacity[Toggle opacity] pour afficher clairement les éléments de filtre.
