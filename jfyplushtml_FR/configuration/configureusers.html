Configurer les utilisateurs

  • Les clients JFY Plus avec des rôles différents sont utilisés pour contrôler la configuration de l’usine.

    • Ajout de nouvelles machines, matières et tôles.

    • Modification des configurations machine.

    • Modification des pièces, tâches et sous-groupes.

    • Modifier / Mettre à jour les tôles d’imbrication, Planifier la tâche.

    • Ajouter / Modifier les utilisateurs.

  • Pserver installe le PC considéré comme le premier utilisateur JFY Plus.

  • JFY Plus utilise l’identifiant Windows comme nom d’utilisateur JFY Plus qui est Site Admin [Droits d’utilisation supérieurs].

  • Installation suivante : le nom d’utilisateur de l’ordinateur d’installation du client JFY Plus est considéré comme Programmer.

Seul Site Admin peut modifier les informations d’un autre utilisateur et modifier le mot de passe.
