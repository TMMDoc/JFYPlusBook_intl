Exporter/importer une matière

Utilisation du format texte

Pour exporter des matières de l’inventaire, accédez à factory → sheets, cliquez surExport/Import, sélectionnez Exporter, choisissez .txt et enregistrez dans le dossier cible.

Ouvrez le fichier .csv exporté, ajoutez une nouvelle matière première et une tôle avec les détails nécessaires, puis enregistrez le fichier.

Pour importer le fichier mis à jour, accédez à factory → sheets → Export/Import, sélectionnez Importer et choisissez le fichier TXT modifié. Les matières et tôles nouvellement ajoutées apparaîtront dans JFY Plus.

Utilisation du format CSV

  • Cliquez sur factory → sheets → Export/Import…​ → ui:EGTag.ExportSheets[Export sheets]. Sélectionnez le fichier CSV et enregistrez-le dans le dossier cible.

  • Ajoutez des tôles et des paramètres de tôle, enregistrez le fichier .csv.

  • À l’aide de l’option Importer des tôles, choisissez le fichier .csv modifié.

  • Les matières et les tôles nouvellement ajoutées apparaîtront dans JFY Plus.

L’importation de matière doit utiliser un fichier texte spécialement conçu qui prend en charge le format suivant. ; Nom-Matière=[MatièrePliage],[Densité],[NomF] ; MatièrePremière-Epaisseur, [Nom]=[Film], [Traitement], [Surface] ; Largeur de tôle, Hauteur, Grain, Desc, NuméroThermique=Qté, [Usine], [Etagère], [Coût], [Commentaires]

Exemple : Aluminium0.5, Aluminium-051500, 1500, Aucun, =112,,,22,2500,1250, Horizontal, =105,,,7,2500,1500, Aucun, =100

Tip
 Pour créer facilement un fichier d’importation, exportez d’abord les matières, puis apportez les modifications ou affichez la structure.
