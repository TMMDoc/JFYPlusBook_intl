Ajouter nouveau

La gestion des tôles est accessible en cliquant sur le bouton factory. La sélection de Sheets vous donnera un aperçu des matières qui sont réglées. En cliquant sur l’une des matières mentionnées sur la gauche vous montrera les tôles, l’épaisseur, les tailles et les quantités qui sont disponibles.

Pour ajouter une nouvelle matière, cliquez sur Add new →*New Material…​* à partir de la page Tôles dans Usine.

Saisissez les paramètres Material, Raw Material, Thickness et matière. Cliquez sur OK.

Voir la matière

Les options d’affichage dans factory → Sheets sont les suivantes :

  • Expand All: Développe toutes les catégories de matière.

  • Collapse All: Réduit toutes les catégories de matière.

  • Show All Materials: Liste toutes les matières premières, quel que soit l’inventaire de tôles.

  • Show Inventory Materials: Liste uniquement les matières premières qui ont un stock de tôles.

L’épaisseur de la matière première indiquée en gras représente la disponibilité de tôle.

Ajouter une tôle

Pour ajouter une nouvelle tôle, cliquez sur Add new → New Sheet à la page Tôle, dans Usine. Modifiez les paramètres de tôle.

La modification ou la suppression de matière/tôle peut être effectuée avant qu’une pièce ne soit affectée à cette matière ou que le layout ne soit imbriqué dans cette tôle.
Il n’est pas possible de changer le nom de la matière à ce stade. Par conséquent, vous devez en créer une nouvelle.
