Flux de travail

Ce chapitre vous donne un aperçu de base des étapes de travail :

1. Configurer le système

Avant de commencer, vous devez configurer JFY Plus comme représentation numérique de votre usine. Vous devez configurer les machines, ajouter des matières, configurer le montage et configurer les utilisateurs.

2. Importer et modifier des pièces

Vous pouvez importer vos pièces dans la bibliothèque de pièces. La technologie est ajoutée instantanément à vos pièces, vous pouvez vérifier cela avec les icônes ci-dessous.

== 3. Contrôler les problèmes de production

Vous pouvez modifier tous les montages avec un clic droit sur la pièce.

== 4. Définir les tâches

Pour combiner des pièces à des ordres de fabrication, sélectionnez-les dans la bibliothèque de pièces et créez une tâche. Créez la tâche avec un clic droit.

== 5. Attribuer des tâches aux machines

Si vous possédez plus d’une machine, vous pouvez sélectionner la machine qui doit effectuer la tâche.

6. Utiliser des filtres pour trouver des pièces

De puissants algorithmes de filtrage sont à votre disposition pour trouver n’importe quelle pièces de vos clients.

7. Libérer la pièce vers les machines

Vous pouvez modifier n’importe quelle tâche à tout moment avant de la libérer vers la machine. Pour produire vos pièces, vous devez assigner des pièces aux machines, le code NC est ensuite transféré à la machine.

== 8. Produire des pièces

Après avoir libéré l’emplacement du dossier par défaut OU en libérant l’emplacement de l’expéditeur de code de vos pièces vers votre machine, elles seront produites et marquées comme terminées.
