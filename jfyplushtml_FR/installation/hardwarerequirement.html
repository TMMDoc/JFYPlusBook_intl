Condition requise pour le port

Serveur JFY Plus : 7371

Le serveur attend les demandes du client sur ce port TCP. Les connexions entrantes doivent être autorisées vers ce port sur l’ordinateur serveur.

Serveur de licence flottante : 5776

Le serveur de licence flottante attend les demandes de licence sur ce port. Les connexions entrantes doivent être autorisées vers ce port sur l’ordinateur serveur de licence flottante (Serveur JFY Plus).

Connexion de la licence au Cloud : 7567

Ce port est utilisé par Flux (logiciel de pliage) et le serveur de licence flottante pour envoyer des demandes de licence au cloud Metamation. La connexion sortante doit alors être autorisée sur ce port pour les activations et les renouvellements de licence.

Condition requise pour l’installation et le support

  • Privilèges administratifs requis pour installer l’application JFY Plus.

  • Outils de diagnostic à distance via le VPN du contracteur ou une connexion sécurisée.
