Modifier le montage

Comment modifier le montage

  • Faites un clic droit sur n’importe quelle pièce qui contient l’instruction de pièce et le nom de la pièce dans son en-tête. Cliquez sur Montages pour basculer sur les premières instances montées. Ou faites un clic droit sur l’instance montée, cela affiche les informations de la machine au niveau de l’en-tête.

    • Afficher : permet de vérifier les instances montées que vous pouvez utiliser.

    • Modifier : permet de vérifier et mettre à jour les instances montées pour faciliter le process de fabrication.

      == Validation du montage après modification

  • Les instances montées modifiées s’ouvrent dans l’application Flux ou MetaCAM. Vérifiez et modifiez le montage manuellement. Lors de l’enregistrement des modifications, la fenêtre de dialogue Modifications non enregistrées apparaît pour confirmer les modifications.

  • Cliquez sur Yes. La Validation du montage est terminée.

    • Si tout est en ordre, « Enregistrer les modifications apportées aux options JFY Plus » s’affiche.

    • Si la pièce montre une erreur de montage, les informations d’erreur respectives seront affichées avec l’option : Ignorer cette erreur et enregistrer les modifications.

    • Corrigez l’erreur : continuez l’édition, ne fermez pas la fenêtre et corrigez l’erreur.

  • Ignorez les modifications et conservez l’ancien résultat - Ignorer les modifications appliquées et fermer la modification.

    == Identifier les pièces avec instances de montage modifiées dans la bibliothèque de pièces

  • Les instances montées vérifiées s’affichent en vert.

    • L’auteur des instances montées est remplacé par le nom de l’utilisateur qui les a modifiés récemment.

    • Modifié par l’utilisateur est défini sur Oui.

    • L’auteur de la pièce est également remplacé par l’instance montée récemment modifiée.

      == Identifier les pièces avec instances de montage en cours de modification dans la bibliothèque de pièces

  • La case de la pièce affiche une icône utilisateur en haut de l’écran qui indique que la pièce est en cours de révision pour modification.

  • Lorsque vous placez le curseur sur la pièce, vous pouvez voir l’utilisateur qui a révisé la pièce, quand et pour quelle machine, etc.

  • Un clic droit sur la même instance montée de la machine n’affiche pas l’option Modifier.
