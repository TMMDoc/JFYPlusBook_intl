Interactive Nesting

Interactive Nesting

  • Select (a) one part or (b)few parts from a Job or (c) one job or (d) multiple Job to be nested → Right-click and Select ui:Str.RunInteractiveNesting[Run Interactive Nesting…​]

  • Part Selection is done.

  • Interactive window switches to select Machine and Sheet selection Tab.

    • Machine is colored with available production load. Green color bar (Scheduled hours) with grey color bar (Available hours) against 8 hours.

    • Under each machine, selected parts are grouped with Raw-Material (Parts/Total Quantity).

  • Suppose a part does not have valid tooling solution for a machine. In that case, that material and part is ignored for that machine.

  • One Raw-Material is checked ON for one machine, then that material wont be available for other machines to plan.

  • Once machine nest plan is done, then we can select the preferred sheet and override sheet parameters like priority, quantity, margin etc.,. In sheet we have filter to show all Sheet, standard or remnant sheets. Click Next.

It is not possible to increase the quantities at this stage. We can either keep same as sheet inventory quantity or reduce it and check nesting results.

  • Update Quantity shows up in Interactive nesting plan window.

  • Pick All or as default all parts are considered for nesting. Or we can manually pick priority parts and send to nesting. We can reduce the part quantity too. Click next.

  • Nesting starts and Part nested in sheet are updated with blue color status. Against the Nest result, machine production load are updated.

  • The nested layouts are drawn to the scale and are displayed in the Nest Results tab. Some parts remain unnested.Appears if some parts are not nested and which stays in earlier Job status.

    • [Reason for nesting may be sheet not available to nest]

  • Interactive Nest result are prefixed with factory → settings → Job→ ui:Str.NestSettings[Nest settings] , Interactive sheet-name prefix.

Auto-Plan Nests are prefixed with Letter P.
