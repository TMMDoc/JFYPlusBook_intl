Export Outputs

Export Outputs - Nest

  • Right-Click any layout which Sent to Machine done and Click Export Outputs.

  • The Export Outputs layout command exports the NC, report and layout files to the respective machine output locations. This command regenerates the NC with current unit set to the layout.

Export output for Nest is not available for Completed layouts and layouts which are waiting in queue for repacking.

Export Outputs - Job

Default Location

  • Right-Click the JOB and Click Export Outputs which exports all Nest (NC, report and csv) to respective machine folder location.

Specify Location

  • We can override the folder location output by clicking Export Outputs → Browse. .

  • Specify Folder where all Nests in JOB are exported (Ignoring the completed Nests).
