Import / Export

Import from csv or xlsx

You could also import jobs in common text formats

  • Click File → Open → Import Jobs, Open Folder dialog appears. Select C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv. Click OK. [Samples\Jobs Folder csv format is already mapped as defaults.]

  • If all Parts exists in the csv location or in lookup folder directories, New Job window appears by listing all parts matching the csv fields.

  • Missing parts are ignored and not appears in the New Job window.

  • Initially Part status is Nascent.

  • Click OK, now parts are imported in Part Library. CAD validation, tooling for bend and cut tooling instance done which can be monitored in the Pmonitor screen.

Create a Blank part for the missing cad files

  • Factory → Settings → Job, under spreadsheet import settings. Check ON Create a blank part fro the missing CAD files

  • If part is missing, then in New Job window which is highlighted in Yellow color.

  • In Job, Missing part will have csv mapped fields but the part state is Nascent ie., blank empty is created.

Do not show the New Job dialog

  • Factory → Settings → Job → Check ON : Do not show the New Job dialog if the spreadsheet has no error

    • If spreadsheet does not have any missing parts, date or any error, New Job is created without New Job dialog prompt

  • Factory → Settings → Job → Check OFF : Do not show the New Job dialog if the spreadsheet has no error

    • Even though spreadsheet does not have any error, New Job dialog prompt for user confirmation.

Prefer Raw-Material column over Material+Thickness

  • Factory → Settings → Job → Check ON : Prefer Raw-Material column over Material+Thickness

    • Spreadsheet material column need to be Raw Material ie., Steel-20

  • Factory → Settings → Job → Check OFF : Prefer Raw-Material column over Material+Thickness

    • Spreadsheet material column need to be Material & Thickness ie., Steel & 2.00

Step 3: Watch Folder using csv or xlsx

  • Factory → Settings → Watch.

    • Check ON : Enable 'Live Folder' & Watch sub-folders also

    • Specify Watch folder UNC folder path or Mapped drive folder path.

    • Spreadsheet Import Settings : Check ON Import spreadsheet from 'Live Folder'

    • From Spreadsheet import option, select Import Jobs

  • Copy C:\Program Files\Metamation\Praxis\Samples\Jobs\C0\C0.csv alone and Paste in Watch Folder.

  • No JOB is created because of CAD file missing

Not Configured : CAD Lookup directory & Recursive CAD Search

  • Factory → Settings → Job → Spreadsheet Import settings.

  • Do not set any CAD lookup Directory and Use Recursive CAD search.

  • Check ON : Create a blank part for missing CAD files.

  • Now Job is created. But all part status is nascent blank part ie., CAD missing.

Configured : CAD Lookup directory & Recursive CAD Search

  • Factory → Settings → Job → Spreadsheet Import settings.

    • Click Edit…​ in CAD Lookup Directory and save the location C:\Program Files\Metamation\Praxis\Samples.

    • Check ON : Use Recursive CAD search.

    • Check ON : Create a blank part for missing CAD files.

  • JOB is created with all parts imported, cad validation and tooling done for all machines.

Export job

The export option will allow you to select different information from the available options and export it out to a csv based file.

To use the job export firstly click on the job icon and then select the export option on the right of the screen. You will now see a dialog box appear where you can click the relevant checkbox to select the fields of information that you would like to export, you can also use the Group By option to group the files by any of the fields that have been enabled.
