Edit Tooling

How to edit tooling

  • Right click on any part that shows the part command and Part name in the header. Click Toolings which switch to first tooled instances. Or Right-Click the tooled instance, shows machine information at the header.

    • View : For checking tooled instances, we can use.

    • Edit : Check and update tooled instances for ease manufacturing process.

Tooling validation after tooling edit

  • Edited tooled instances opens in Flux or MetaCAM application. Do manual tooling check and edit, while saving the changes - unsaved changes dialog appears for confirmation of the edit changes.

  • Click Yes. Now again Tooling Validation done.

    • If all OK, Save changes to JFY Plus options appears.

    • Part has tooling error, respective error information will be shown along with option : Ignore this error and save the changes.

    • Correct the Error : Continue editing, don’t close the window, and do the error correction.

  • Skip the edit changes and retain the old result - Discard the changes applied and close editing.

Identify Edited tooled instances parts in part library

  • Checked-in tooled instances color is green.

    • Tooled instances Author is updated with respective user who edited recently.

    • User-edited is set as Yes.

    • Part author also changed to recent edited tooled instance.

Identify currently Editing tooled instances parts in part library

  • Part tile shows a user icon at the top representing this part is checkout for editing.

  • Placing cursor over the part shows user who checked out, when and which machine etc,.

  • Right-Click over same machine tooled instance will not show edit option.
