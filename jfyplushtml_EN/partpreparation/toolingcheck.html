Tooling Check

Tooling Check

Re-import

  • Right-Click any part and Click Re-import…​

    • All selected parts tooling instances [all machines] either auto-tooled or manually edited tooled and output files are removed first. [CAD, Bend and Cut outputs]

    • Parts are again imported with CAD validation, tooling for all machine instances.

    • Depending upon part tooling result, output files are created.

Re-tool

  • Right-click any part and Click Re-tool…​

    • All Machines Checked ON with all machines sorted in group Laser, Punch, Panel and Press Brake.

    • Click OK → Auto-tooled or Manually edited selected tooled instances are removed along with Output location file [Bend and Cut outputs].

    • Retain existing outputs :

      • Checked OFF : All selected parts against selected machine tooling instances of Bend and Cut Outputs were removed first. Depending upon re-tooling result, output file are exported.

      • Checked ON : Instead of removing all output of the part. In Batch, first few set part collection, Bend and Cut Outputs were removed and Re-tooled those parts.

Use of Re-tool
====

  • Change in cam settings of one or two machines, then we can select those machine and Re-tool.

  • Suppose change in LTT or punch tool lost or bend tool lost, then update machines. Re-tool those part of those machines alone. ====

Ignore Errors

  • Suppose a tooled instances have warning which can be override against the auto-tooled result by clicking Ignore Errors .

    • Clicking Ignore Errors will export respective Outputs.

Important Note
====

Please be cautious while Ignore Errors against any Error parts which may cause damage to machine or accidents. ====

Check Tooling

  • Suppose a bend tool lost or any CAM settings change, no need to re-tool those bend tooled instances.

    • Use Check Tooling, this will do tooling validation alone.

    • If tooling validation fails, that tooled instance status changes. Similarly, Output file will be removed according to the status.

    • Check Tooling will flag manually override Errors by Ignore Errors and flag the part to error state.

Suppress/Unsuppress Tooling

  • Suppose a particular machine is down or particular part not to process in that machine, an option Suppress Tooling is used to avoid nest getting prepared for this part against this machine.

    • Place the mouse over the tooled instances after suppress tooling will show Tooling suppressed by <User name> on <Date Time>.

    • Suppress Tooling will remove the respective Output file.

  • Unsuppress Tooling return back the Tooling status to suppressed state. Also respective output file will be exported again.
