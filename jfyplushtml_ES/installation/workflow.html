Flujo de trabajo

Este capítulo le ofrece una descripción general básica de los pasos de trabajo:

1. Configurar el sistema

Antes de comenzar, debe configurar JFY Plus que sea la representación digital de su fábrica. Debe configurar máquinas, añadir material, configurar útiles y configurar usuarios.

2. Importar y editar piezas

Importa sus piezas a la biblioteca de piezas. La tecnología se añade instantáneamente a las piezas, lo que puede comprobar con los iconos que se muestran a continuación.

== 3. Comprobar problemas de producción

Puede editar todo el utillaje haciendo clic con el botón derecho del ratón en la pieza.

== 4. Definir tareas

Para combinar piezas con tareas de producción, estas se seleccionan en la biblioteca de piezas y se crea una tarea. Cree la tarea haciendo clic con el botón derecho del ratón.

== 5. Asignar tareas a máquinas

Si tiene más de una máquina, puede seleccionar la máquina para producir la tarea.

6. Utilizar filtros para encontrar piezas

Hay potentes algoritmos de filtro para encontrar cualquier pieza de sus clientes.

7. Liberar la pieza a las máquinas

Puede cambiar cualquier tarea en cualquier momento antes de liberarla en la máquina. Para producir sus piezas, debe Asignar pieza a máquinas, luego el código CN se transfiere a la máquina.

== 8. Producir piezas

Después de liberar la ubicación de la carpeta predeterminada O liberar la ubicación de Code Sender Location en su máquina, se producen y se marcan como completas.
