Requisitos de puertos

JFY Plus Servidor : 7371

El servidor espera las solicitudes del cliente en este puerto TCP. Las conexiones entrantes deben permitirse en este puerto en el ordenador servidor.

Servidor de licencias flotantes : 5776

El servidor de licencias flotantes espera las solicitudes de licencia en este puerto. Se deben permitir las conexiones entrantes a este puerto en el ordenador servidor de licencias flotantes (servidor JFY Plus).

Conexión de licencia a la nube : 7567

Este puerto es utilizado por Flux (software de plegado) y el servidor de licencias flotantes para enviar solicitudes de licencia a la nube de Metamation. Por lo tanto, se debe permitir la conexión saliente en este puerto para las activaciones y renovaciones de licencias.

Requisitos de instalación y soporte

  • Privilegios administrativos necesarios para instalar la aplicación JFY Plus.

  • Herramientas de diagnóstico remoto a través de Contractor VPN o una conexión segura.
