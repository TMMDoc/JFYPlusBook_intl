Watch settings

Enable 'Live Folder' - Establezca esta opción para habilitar la carpeta en vivo.

Watch this folder - Establezca esta como la ubicación la que Praxis supervisará.

Watch sub-folders also - Habilite esta opción para que Praxis también verifique las subcarpetas dentro de la ubicación indicada anteriormente.

After File Import

After import - Después de importar el archivo, el usuario tiene dos opciones, el archivo se puede eliminar de la ubicación o mover a una ubicación diferente.

Move files to this folder - Si la opción anterior es mover el archivo a una ubicación diferente, introduzca aquí la ubicación a la que se debe mover el archivo.

Move spreadsheets with errors to this folder - Si se descubre que una hoja de cálculo tiene errores, se puede mover a una ubicación diferente que se puede indicar en esta sección.

Spreadsheet import settings

Import spreadsheet from 'Live Folder' - Habilite esta opción para permitir que Praxis importe hojas de cálculo que se encuentran en la ubicación de la carpeta activa.

Do not import spreadsheet with errors - Habilite esta opción para que Praxis no importe hojas de cálculo con errores. Se pueden mover a una ubicación diferente, como se indicó anteriormente, para ser editadas.

Spreadsheet import option - Esta opción puede establecer un tipo de importación predeterminado que puede ser cualquiera de las siguientes opciones según su licencia disponible.

  • Importar tareas

  • Importar pedidos

  • Importar grupos de módulos
