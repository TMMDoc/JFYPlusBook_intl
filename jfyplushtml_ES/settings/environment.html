Environment settings

Program default units - Ver JFY Core ajustes. Decimal places to display - Ver JFY Core ajustes. Decimal accuracy in export/import - Este valor es para establecer cuántos decimales deben ser exactos al importar o exportar.

Display part toolings in groups - Utilice esta opción para seleccionar el grupo de utillaje para mostrar el estado del utillaje en la biblioteca de piezas.

Captions for command-bar icons - Ver JFY Core ajustes. Display Job status labels - Al habilitar esta opción, se mostrará el número de piezas que están dentro de una tarea al final de la barra de estado de la tarea. Cuando esta opción está desactivada, se elimina el recuento de piezas y el color para mostrar el proceso permanecerá.

Display nested parts by job-order color - Esta opción mostrará cada parte diferente dentro de una tarea como un color diferente. Al desactivar esta opción, todas las piezas se mostrarán en el mismo color gris.

Display the duration in time-span format (HH:MM:SS) - Marque esta opción para mostrar la duración en formato HH:MM:SS. Cuando está habilitado, la duración en los detalles del diseño y los informes de tarea/diseño se muestran en el formato seleccionado. Debido a las limitaciones de espacio, la duración se muestra en un formato de segundos más compacto, en los mosaicos de diseño.
