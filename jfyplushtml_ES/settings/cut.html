Cortar

Output settings

Save cut outputs- Los resultados del mecanizado solo se guardan si el mecanizado es correcto y si estos ajustes de exportación están habilitados y configurados. Active esta opción para que se guarde el CN de corte.

Output tooling simulation preview - Habilite esta opción para generar la vista previa de simulación de utillaje junto con las otras salidas. Esta es una breve animación GIF para el utillaje láser/de punzonado.

Cut output destination - Este es el destino donde se guardarán el CN y los informes.

MetaCAM settings

Lookup MetaCAM materials by raw-material names (Enable this for DMM like integrations) - Cuando este estilo está habilitado, Praxis genera páginas con nombres de materias primas y las asigna a las piezas en el momento del mecanizado automático.

MetaCAM sync folders - Esta configuración de corte se utiliza para añadir carpetas MetaCAM adicionales que deben sincronizarse junto con config.dat cuando se utiliza la orden config-push Monitor. Config-push guarda los archivos locales en el repositorio de Praxis, que se descarga en los otros equipos cuando se (re)inicia MetaCAM. El ordenador del motor debe sincronizarlo automáticamente cuando se realice el mecanizado automático para una máquina MetaCAM, aparte de las Herramientas, algunos de los otros candidatos específicos de la instalación son las carpetas Etc, Informes y Máquinas.

DMM config sync location - Esta es la ubicación para que se establezca la sincronización de configuración de Dynamic Machine Management.

Flux Settings

Enable multi-pass nesting - Esto permitirá que el láser corte varias veces en el mismo contorno (preferido para material más grueso).
