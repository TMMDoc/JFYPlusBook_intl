Notificaciones

Notification settings

Enable email notifications - Esta opción se utiliza para habilitar y configurar la notificación por correo electrónico.

Notification recipients - Introduce los datos del destinatario, para introducir más de un destinatario utiliza un espacio o una coma para separarlos.

Send messages in batch - Esta opción se utiliza para combinar varios mensajes y enviarlos por lotes. Para evitar mensajes grandes, los mensajes enviados dentro de una duración finita se combinan.

Attach tooling simulation preview where applicable (Warning: Messages may become too large) - Esta opción añadirá una vista previa de simulación de utillaje a la notificación por correo electrónico.

Mail settings

SMTP server - La dirección del servidor de correo electrónico SMTP se puede encontrar en la sección de cuenta o configuración del cliente de correo del usuario.

SMTP port - Esta configuración es el estándar básico que utilizan los servidores de correo para enviarse correos electrónicos entre sí a través de Internet.

Transport protocol -

Server needs authentication - Habilite esta opción para que el servidor sea validado.

User-name - Introduzca aquí el correo electrónico del destinatario.

Password - Introduzca aquí la contraseña del destinatario.

Test…​ - Utilice el interruptor de prueba para comprobar si un correo electrónico se ha entregado correctamente al usuario. Se mostrará un mensaje de error, como en la imagen de abajo, si el correo de prueba falló.

La notificación por correo no funciona en modo interactivo. Por lo tanto, editar una pieza, aplicar un cambio y guardar no generaría una notificación que se envía al usuario, incluso si hay un error de utillaje/diseño en la pieza que se está guardando.
Debido a la naturaleza de la comunicación por correo electrónico, la entrega de correo puede ser mucho más lenta y, por lo tanto, se realiza en segundo plano. Las notificaciones pueden perderse si el motor se detiene/apaga antes de que se entreguen todos los mensajes.
