通知

Notification settings

Enable email notifications -此选项用于启用和配置电子邮件通知。

Notification recipients -输入收件人详细信息，若要输入多个收件人，请使用空格或逗号分隔。

Send messages in batch -此选项用于组合多个消息并批量发送。为避免大量消息，在有限持续时间内发送的消息将被合并。

Attach tooling simulation preview where applicable (Warning: Messages may become too large) -此选项将向电子邮件通知添加加工模拟预览。

Mail settings

SMTP server - SMTP电子邮件服务器地址可以在用户邮件客户机的帐户或设置部分找到。

SMTP port -此设置是邮件服务器用于通过互联网相互发送电子邮件的基本标准。

Transport protocol -

Server needs authentication -为要验证的服务器启用此选项。

User-name -在此处输入收件人的电子邮箱。

Password -在此处输入收件人的密码。

Test…​ -使用测试开关检查电子邮件是否已成功发送给用户。如果测试邮件失败，将显示错误消息，如下图所示。

邮件通知在交互模式下不起作用。由此，编辑零件、应用更改和保存不会生成发送给用户的通知——即使正在保存的零件中存在加工/设计错误。
由于电子邮件通信的性质，邮件传递可能会慢得多，因此是在后台完成的。如果在所有消息传递之前停止/终止引擎，通知可能会丢失。
