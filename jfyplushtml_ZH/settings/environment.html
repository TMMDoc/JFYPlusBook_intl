Environment settings

Program default units -请参阅JFY Core设置。 Decimal places to display -请参阅JFY Core设置。 Decimal accuracy in export/import -此值用于设置导入或导出时精确的小数位数。

Display part toolings in groups -使用此选项选择用于在零件库中显示加工状态的加工组。

Captions for command-bar icons -请参阅JFY Core设置。 Display Job status labels -启用此选项将在套料状态栏的末尾显示套料内零件的数量。禁用此选项后，零件计数将被删除，显示过程的颜色将保持不变。

Display nested parts by job-order color -此选项会将套料中的每个不同零件显示为不同的颜色。禁用此选项后，所有零件将以相同的灰色显示。

Display the duration in time-span format (HH:MM:SS) -选中此选项以HH: MM: SS格式显示持续时间。启用后，布局详细信息和套料/布局报告中的持续时间将以所选格式显示。由于空间限制，持续时间以更紧凑的秒格式显示在布局图块上。
