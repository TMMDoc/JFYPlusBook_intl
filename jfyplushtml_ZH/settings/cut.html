切割

Output settings

Save cut outputs -只有当加工正常并且这些导出设置已启用和配置时，才会保存加工结果。启用此选项以保存切割NC。

Output tooling simulation preview -启用此选项可生成加工模拟预览以及其他输出。这是激光/冲裁加工的简短GIF动画。

Cut output destination -这是要保存的NC和报告的目标位置。

MetaCAM settings

Lookup MetaCAM materials by raw-material names (Enable this for DMM like integrations) -启用此样式后，Praxis将按材料名称生成页面，并在自动加工时将其分配给零件。

MetaCAM sync folders -此切割设置用于添加额外的MetaCAM文件夹，当使用配置推送监视器命令时，这些文件夹应与config.dat同步。配置推送将本地文件保存到Praxis存储库，该存储库在MetaCAM（重新）启动时下载到其他计算机上。除了加工之外，当MetaCAM机床的自动加工完成时，引擎计算机应自动同步它，其他一些安装特定的候选者是Etc、报告和机床文件夹。

DMM config sync location -这是要设置动态机床管理配置同步的位置。

Flux Settings

Enable multi-pass nesting -这将使激光能够在同一轮廓上切割多次（首选用于较厚的材料）。
