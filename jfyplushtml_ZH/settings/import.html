导入

Output settings

Save flat pattern -为要保存的导入零件的平面图案启用此选项。

Format for flat pattern -请参阅JFY Core设置。 Flat pattern destination -这是保存平面图案的位置。

Import settings

Units for DXF files -请参阅JFY Core设置。 Material lookup keys -可以使用逗号分隔符分配多个查找条目。导入零件时，Praxis使用这些关键词来搜索材料值。然后使用搜索值来解析材料。原始值存储在零件“杂项属性”中。

Revision lookup keys -导入零件时，将原始零件文件（正在导入的零件文件）的修订值与存储库中的现有修订值进行比较。如果修订值相同，则放弃导入。DXF文件现在也支持自定义字段映射。对于DXF文件，自定义字段数据从图形或PPI附件中的文本图层中提取。

Part name keys -此选项的工作原理类似于材料和修订查找关键词。设置一个或多个零件名称关键词，以使用PPI零件名称而不是文件名作为零件名称。

Thickness lookup keys -可以使用逗号分隔符分配多个查找条目。导入零件时，Praxis使用这些关键词来搜索厚度值。然后使用搜索值来解析厚度。原始值存储在零件“杂项属性”中。

Thickness comparison fudge -用于在导入零件时查找材料。Praxis尝试在解析材料名称和厚度后解析材料。有关更多信息，请参阅以下[Thickness fudge]主题。

Identify shapes after a flat is imported -导入零件时，可以自动识别和识别零件内的内部形状。

Shape identification fudge -形状识别模糊值可以设置为：

  1. 全局级别 -此值用作所有形状的默认值。

  2. 单个形状级别 -可以通过在单个形状级别设置模糊值来覆盖默认值。在“形状库”对话框中选择形状会显示模糊值。默认情况下，此值设置为0，默认全局值用于此类形状。

Rotate flat horizontally -一般来说，切割机台面是矩形的，宽度更大。为提高加工的可行性，请在导入零件后沿零件宽度旋转零件。

Part rotation threshold -将其设置为允许旋转的最小数量。任何小于设定值的零件都不会旋转。

Misc. settings

Enable pre-import external tools -启用预导入外部模具导入选项以启用Creo读取器。Praxis上传已丢弃零件的预提取系列实例。请注意，实例提取是在零件上传期间执行的（作为预导入操作），因此必须在上传零件的计算机上安装Creo软件。

Ignore drawing colors in Flat thumbnails for clarity -启用此选项可生成更清晰的灰度缩略图。为了图像清晰度，将忽略图层颜色和文本实体。

Use only one Engine for all import tasks -启用后，只有一个引擎核心用于重要任务。

厚度模糊值

当所需的板材厚度在库存中不可用时，通过比较库存中板材的厚度值，*Thickness comparison fudge*选择最接近的板材厚度值。例如，我们假设：

  1. 我们的板材零件厚度为1.6毫米，材料为钢。

  2. Thickness comparison fudge 值为0.1毫米。

  3. 库存的钢板厚度为1.5毫米、1.55毫米和1.7毫米。

    image::SheetThickness.png[Sheet inventory,621] JFY Plus将*Thickness comparison fudge*值（0.1毫米）与导入物品（1.6毫米）进行比较后，选择厚度值最接近的板材。 在这种情况下，将选择1.55 mm。
