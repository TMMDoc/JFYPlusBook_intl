套料

Output settings

Save layout NC and reports -为要保存的布局NC和报告启用此选项。

Save layout file -为要保存的布局文件启用此选项。

Output simulation preview in reports -启用此选项可生成加工模拟预览以及其他输出。这是激光/冲裁加工的简短GIF动画。

Do not delete program outputs on layout/job completion -当套料/布局被标记为已完成时，启用此选项将跳过程序输出的删除操作。

Nest settings

Mix parts from multiple Jobs -支持混合模式排样。多个套料的零件应排样在一起，以提高填充效率。

Do not auto-release layouts -这将停止可用于生产的布局，直到所有布局都处于已发布状态。

Re-nesting packing cut-off -此值用于确定排样布局是否保存在批准队列中，并在计划新零件或将其发布用于生产并将相关输出导出到输出文件夹时再次重新排样。

Interactive sheet-name prefix -此部分用于支持交互式排样，用户可以更好地控制正在排样的零件和排样结果。对于交互式排样布局，请使用与自动排样不同的命名方案。

Update sheet inventory on nest (un)release -如果启用此选项，则在向机床发送排样时将更新板材库存。此外，从机床中召回排样将使板材返回库存。

Automate remnant inventory management -选择此开关后，软件将自动进行残留物库存管理。如果未选中，残留物管理将变为交互式。

Auto-refresh Out-of-date layouts -选中后，软件会在计算出其切割解决方案后刷新链接到某个零件的所有过期布局。

Automate twin-line nesting -选中此选项可启用TwinLine自动化装置。

Pull workflow settings

late:SH.UsePullWorkflow [Use 'Pull workflow'] -此选项对于需要高度自主规划和排样系统的用户非常有用。启用后，Praxis按需计划排样。

Nests per station -此值用于设置每个工作站应创建多少个布局。

Maximum sheets per nest -此值用于控制针对唯一排样的最大板材计数。因此，如果订单包含大零件（填充整个板材），需要大数量（例如500个）- Praxis将不再在拉动模式下创建具有500个副本的单个排样。相反，它为每个排样创建10个副本，其中10是最大值。

Bend programs per station -此值用于设置每个工作站应创建的折弯程序数。

Mark the task completed, when the program outputs are removed -启用此选项，以便在从目标输出位置中删除输出时将布局标记为完成。

Spreadsheet import settings

CAD lookup directories - Praxis将尝试从零件库加载导入的CSV中提到的零件条目。但是，如果该零件在零件库中不可用，则它将尝试根据在此对话框中设置的目录顺序从查找目录中获取零件。

Use recursive CAD search -启用此选项可重复搜索以获得连续结果。

Create a blank part for the missing CAD files -如果Praxis库或查找目录中不存在零件，Praxis将为这些不可用的零件创建一个毛坯。

Do not show the New Job dialog if the spreadsheet has no error -如果CSV条目是干净的，即数据库中的数据没有不匹配，或者所有字段都正确列出，则Praxis将不会显示套料概览窗口，并将在导入或删除CSV后立即加工套料。

Prefer Raw-Material column over Material + Thickness -支持的电子表格类型：

  1. 模式1：PartName、Quantity、DueDate、Priority、RawMaterial

  2. 模式2：PartName、Quantity、DueDate、Priority、Material、Thickness

  3. 模式3：PartName、Quantity、DueDate、Priority、Material、Thickness、RawMaterial

如果未选中此选项，则模式1将被视为默认电子表格类型。我们还支持模式3，以适用CSV格式为模式3的情况。

如果选中此选项且CSV为模式2，则Praxis将从CSV的材料和厚度字段中解析材料。

Spreadsheet output settings

Export Job/Layout completion report (.csv) -选中此选项可在布局或套料完成时启用CSV导出。完成的报告保存在工厂→设置→报告设置中指定的报告目标位置。
